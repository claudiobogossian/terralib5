Property Converter

This interface allows the user to define how a property should be converted. A series of converters are provided, allowing the user to perform simple conversions or complex conversions using more than one property.

The Origin Properties frame shows all properties of the dataset. Use the property check box to choose which properties will be involved and then select the converter to be used.

The Converter frame displays all types of conversion supported. The types are:

  • GenericAttributeConverter: Simple conversion between data types.
  • XYToPointConverter: Converts two basic attributes for geometric type - point. An attribute should identify the point's X coordinate and the Y coordinate of the other point.
  • XYZToPointConverter: Converts three basic attributes for geometric type - point. Besides the x and y attributes, an attribute must identify the Z component.
  • XYMToPointConverter: Converts three basic attributes for geometric type - point. Besides the x and y attributes, an attribute must identify the M component.
  • XYZMToPointConverter: Converts four basic attributes for geometric type - point. Besides the x and y attributes, two attributes must identify the Z and M components.
  • PointToXConverter: Converts a geometric attribute of type point, generating a basic attribute with the X coordinate.
  • PointToYConverter: Converts a geometric attribute of type point, generating a basic attribute with the Y coordinate.
  • PointToZConverter: Converts a geometric attribute of type point, generating a basic attribute with the Z coordinate.
  • PointToMConverter: Converts a geometric attribute of type point, generating a basic attribute with the M coordinate.
  • TupleToStringConverter: Converts a set of attributes for a single attribute of type string.

The new property to be generated is defined in the Target Property frame, where the data type, property name and other parameters are filled. To assist in this process use the Property Interface.
