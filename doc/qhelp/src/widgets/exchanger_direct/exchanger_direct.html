Layer Exchanger


The Layer Exchanger interface is used to export one TerraLib layer to a external source.  The list of output data sources types may vary depending on the active plugins (PostGIS, OGR, etc).

It is accessible through:

    Layer Explorer >  Right Clicking over a Layer-name > Exchange... (used to export a specific layer)

    Tools > Data Exchanger > Layer Exchanger... (list of all layers will be available)

To export a layer is required the following steps:
  1. Select the Input Layer .
  2. Select the Output Data Source Type. The next fields may vary according to the selected Data Source Type. 
  3. Inform the Output Data Source  (it is the database name managed by the data source type) or
  4. Inform the Data Set Name  and the location path where the data will be exported.
  5. Optionally check Create Spatial Index.  It is only available if it is supported by the data source type.
At the end of the process a successful message or a error indication will be displayed.

NOTE: It is possible to add a new Output Data Source to store the exported data by clicking on the database icon showed. Usually it is used when there is no data source available to choose. The  Data Source Explorer interface will be showed in this case.
