Tool Bar Configurations Settings

The Tool Bar Configurations Settings is used to change the default configurations of the tool bar.  At this example there are two tool bars defined (named File Tool Bar and Map Tool Bar)  and the Actions selected for each of them are showed togheter at the main window.  

To see  the Tool Bar Names click on View or right click over an empty area at the Tool Bar. Observe that the Tool Bar Names do not appear at the main window.  It is only a name to be referenced at the User settings Customizing.

It is possible to add (+) a tool bar by giving a new tool bar name and selecting some Actions to group at this new tool bar. For example, click on (+) and give the new tool bar name, por example Graphic. Note that all available Actions will be unclicked, then click on one or more Actions related with graphic, for example: Histogram, Scatter, Chart. Click on Apply and see the new tool bar at the the righ side of the main window below.
