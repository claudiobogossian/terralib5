Dataset Selector

The Dataset Selector shows all datasets available for the data source chosen in the  Data Source Selector  (PostGIS at this example).

To select one or more datasets just check the corresponding box and press Select button.

Before the selection, it is possible to see a  Map Preview or Data Preview of each dataset (check the box). Then,  click over the dataset name on the list (or check the dataset corresponding box) and a preview will be showed. 



The Select  will finish this step and all datasets with the box checked will appear in the TerraView main window on the Layer explorer. The behavior of  the Dataset Selector is the same to all data source types.
