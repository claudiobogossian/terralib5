Data Table

Data table is a ​​organized set of values , composed of columns, which are identified by their name, and rows. Each cell represents the intersection of a column with a row. This section contains functions used to display and manipulate data table such as statistics, hide column and sort data. It will explain how to use each one.

First of all, click the right mouse button on the layer you want to display the table and click on Show Table. As shown in the following figure.



Then click the right mouse button on the column you want to perform some function of the data table. As shown in the following figure.



The functions supported by the table are:

  • Basic features:  Line Selection, Multiline selection, Promote selected rows, Columns selection, Change order of columns.
  • Hide column: Hide a specific column of the data table.
  • Show hidden column / Show all columns: Shows a specific hidden column or all columns that are hidden.
  • Reset columns order: Reorders the columns in their original positions.
  • Sort data ASC / Sort data DESC: Sort the table data based on the selected column in ascending or descending order
  • Statistics: Calculates the statistical summary for the selected column.
  • Add column: Adds a new data table column (Under development).
  • Remove column: Remove any column of the data table (Under development).
