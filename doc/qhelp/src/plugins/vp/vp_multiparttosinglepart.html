Multipart to Singlepart

This operation split a Multipart geometry to Singlepart geometry.

The operation receive a layer that contains multi-geometries and split this geometries to single-geometries. There are options to process all geometries or only selected.

As output, the operation creates a dataset with the result and provides the option to create a layer in the tree.

   Plugins > Vector Processing > Multipart to Singlepart...

  1. Select the Input layer to apply the operation.
  2. Select if the process will occur for all geometries or only in selected
  3. Inform the Output Repository by clicking on:
    1.  to select the output directory and also inform the new layer name to store the result, or
    2.  to select the Data Source.
  4. Inform the new Layer Name to store the operation result, if the repository is a data source.
  5. Click OK to perform the operation or Cancel to close the interface.
Usage Tips:

The operation result can be stored in different TerraLib data sources. Check data source selector for more details on how to select a data source.

The result geometries will receive all attributes equal to original geometry.
