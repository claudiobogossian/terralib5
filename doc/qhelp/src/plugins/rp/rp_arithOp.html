Arithmetic Operations
The Arithmetic Operations allows the user to perform operations on one or more images.  In addition to the operations that can be made between the bands of the images, it is also possible to apply gain and offset to the images. Some standard operations are defined in the interface:
  • Gain * Image A + Offset
  • Gain * (Image A + Image B) + Offset
  • Gain * (Image A - Image B) + Offset
  • Gain * (Image A * Image B) + Offset
  • Gain * (Image A / Image B) + Offset
  • (Gain * (Image A - Image B) / (Image A + Image B)) + Offset
It is accessible through:
   Raster Processing > Arithmetic Operations... (list of all raster layers will be available) 

This wizard consists of the following steps:

    Wizard Page 1 - Selection of the layers to execute the operation (Layer Search)

  1. On the List of Layers select the raster layers to apply the operation. 
  2. Optionally use Filter By Name field giving part of the layer name to help find the layer in the list.
  3. Press Next  to go to next step or Cancel to close the dialog. 

    Wizard Page 2 - Arithmetic Operations parameters

  1. Select which layers will form part of the operation and their respective bands.
  2. Select the desired operation.
  3. Set the optional parameters.
    • Gain value,
    • Offset value
    • Normalize option ( the values of the resulting image will be normalized ).

    Wizard Page 3 - Output information

  1. Raster Info - First press and inform the folder where the resulting file will be saved.
  2. Name - inform the raster name.
  3. Extra Parameters - if there are some, see the details on how to inform then here.
  4. Press Finish to save the resulting contrasted raster or Back to go to the previous wizard page.
Hint: The resulting image will be added as a new layer at the TerraView project
