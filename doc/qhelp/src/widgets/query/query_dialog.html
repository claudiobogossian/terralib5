Query Dialog

The Query Dialog is a interface, which has the functionality to create a  selection on a layer, given a restriction clause.

It is accessible through:

    Layer Explorer > Right Click over an layer name > Query...

    Layer > Query...


This interface consists of the following steps:
  1. Select the input layer.
  2. Defines the restriction clause.
  3. Layer Selection options:
    • Add: Transforms the query result in the layer selected items, adding the query results to layer selection.
    • New: Transforms the query result in the layer selected items and remove previous layer selection.
    • None: The result will be just painted over the map
    • Highlight:  Sets the drawing color for items resulting from the query.
