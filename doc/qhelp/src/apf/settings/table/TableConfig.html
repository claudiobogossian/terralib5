Table Configurations Settings

The Table Configurations Settings is used to change the default settings related with the table display. The default color for all lines in a table is white. It is possible to have different colors for alternate lines as showed at the Preview section of the interface with the default gray color.

It is accessible throw:

Tools > Customize... > Table

To apply the default gray color to alternated lines just click on Apply.

To change the default gray color to alternate lines, do:

  1. Click  on Primary Color,
  2. Choose the color and check the Preview.
  3. Press Apply to see the table with the chosen color.  
  4. To set the default gray color again just take out the check box and check again the Alternate Row Colors.
  5. To set all lines to the default white color just take out the check box of Alternate Row Colors.
Other options will be available soon.
