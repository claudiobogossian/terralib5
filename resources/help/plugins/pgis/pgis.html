Defining PostGIS as a Data Source??

The PostGIS data source connection is defined by a specific set of parameter.  It is accessed through the Data Source Selector.

To define PostGIS as a data source,  fill the Title and the Connection Information needed by PostGIS including the Database Name.
Open will make this connection available at Data Source Selector with the same title specified here.




It is possible to Test the connection before open it. Also there is Advanced Connections Options to change same default parameters. It is possible to hide spatial metadata tables (spatial_ref_sys, geometry_columns) , to define new connection pooling parameter and so on.
