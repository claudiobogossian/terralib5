Vector to Raster

This operation aims to create a new raster layer based on attributes values from a vector layer. Each attribute of the vector layer is a band of the resulting raster.

It is accessed through:

    Plugins > Attribute Fill > Vector to Raster...
  1. Select an Input  vector layer to apply the vector to raster operation.
  2. Select one or more attributes from layer to set the band(s) of raster layer.
  3. Set the resolution or the number of Columns/Rows of the raster result. Optionally set the Dummy value.
  4. Inform the Output Repository by clicking on to select the output directory and also inform the new layer name to store the result.
  5. Click OK to perform the operation or Cancel to close the interface.
