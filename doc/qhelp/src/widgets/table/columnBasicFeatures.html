Column Basic Features

The basic features to manipulate columns in a data table are listed bellow. To access then right click over the column name.
  • Hide column
  • Show hidden column / Show all columns
  • Reset columns order
  • Sort data ASC / Sort data DESC
  • Histogram
  • Statistics
  • Add column (Under development).
  • Remove column  (Under development).
  • Rename Column
  • Change Column Type
  • Change Column Data
  • Save edition

Hide Column

  1. To hide a column from data table,  right-click over the column header and select the option Hide column.

Show Hidden column/ Show All Columns

  1. To show a specific or all hidden columns  from a data table,  right-click on any column and hold the mouse on Show hidden column option to expand the options menu with the name of the columns that are hidden, then click on the column name you want to display in the table. If there is no hidden column, this function is disabled.

Reset columns order

  1. To reorder the columns names in their original positions,  right-click on any column header, then click on Reset columns order.

Sort data ASC / Sort data DESC

  1. To sort the data in ascending order based on the selected column, right-click on the column desired, then click on Sort data ASC.
  2. To sort the data in descending order based on the selected column,  right-click on the column desired, then click on Sort data DESC.

Histogram

  1. To generate and show a histogram for the selected column,  right-click on the column desired, then click on Histogram.

Statistics 

  1. To view basic statistic summary on a selected column,  right-click on the column, then click on Statistics.

Add Column

  1. To add a new column in the data table, right-click over the columns header, then click on Add Column and fill the parameters of the new column.

Remove Column

  1. To remove a column of a data table, right-click over the column name, then click on Remove Column.

Rename Column

  1. To rename a column, right-click over the column name, then click on Rename Column and enter the new name.

Change Column Type

  1. To change the column type, right-click over the column name, then click on Change Column Type and enter the new type.

Change Column Data

  1. To change the column data values, right-click over the column name, then click on Change Column Data and fill how the values should be changed.
