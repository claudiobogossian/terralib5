Data Exchanger

The Data Exchanger is a component build as wizard interface, which has the functionality to exchange data between data sources. Can be defined as an exporter generic, allowing the choice of data types to be exported or allowing the change of data types in case of incompatibility between sources.

The Data Exchanger is a tool used to exchange data between data sources. It is viewed as a generic exporter, 
allowing for each dataset to choose which attributes to export. Also it allows to change the attribute type if there is any incompatibility of data types supported among the data sources involved.

It is accessible through:

    Tools > Data Exchanger > Advanced... and follow the wizard interface pressing Next after each step.
  1. Select the data source where the data is stored.
  2. Select the datasets you want to transfer to another data source.
  3. Select the target data source.
  4. On Selected datasets frame there is a list of datasets to be exported. For each dataset selected you can change or provide more information on how the datasets will be transferred and mapped to the target data source, as explained below:
    1. Name, Title and SRID (changing this will not cause data reprojection ).
    2. Observe the Properties frame.  The Status column of each property shows if the default conversion is supported  between the data sources involved. A check icon  is used to represent that the property was properly adapted, otherwise this icon  is shown. It is possible to add  attributes, Property Converter Interface, or remove  attributes listed.
    3. On the Constraints/Indexes frame it is possible to add  constraints and indexes to the new dataset. The existing ones are listed  and can be removed . See Constraints and Indexes Interface  for more details.
  1. After all click on Commit to export the dataset to the target data source.
  2. Check the Data exchange summary  that will report the success or failure of the whole process.
