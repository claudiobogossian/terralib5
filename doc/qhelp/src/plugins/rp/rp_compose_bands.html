Compose / Decompose Bands
The Compose / Decompose Bands operations allows the user to compose a single raster with bands from different raster and also decompose a raster in separate bands.

It is accessible through:
   Raster Processing > Compose / Decompose Bands... (list of all raster layers will be available) 

This wizard consists of the following steps:

    Wizard Page 1 - Selection of the layers to execute the operation (Layer Search)

  1. On the List of Layers select the raster layers to apply the operation. 
  2. Optionally use Filter By Name field giving part of the layer name to help find the layer in the list.
  3. Press Next  to go to next step or Cancel to close the dialog. 

    Wizard Page 2 - Compose / Decompose Bands

  • Compose Tab
  1. Select which Bands will form part of the composition. Use the  and buttons to add or remove items to list.
    • Band: Select the band from layer to be used in composition operation.
    • Layer/Raster: Select the layer with desired band.
  2. Select the desired interpolator.
  • Decompose Tab
  1. Select which bands will be decomposed.

    Wizard Page 3 - Output information

  1. Raster Info - First press and inform the folder where the resulting file will be saved.
  2. Name - inform the raster name.
  3. Extra Parameters - if there are some, see the details on how to inform then here.
  4. Press Finish to save the resulting contrasted raster or Back to go to the previous wizard page.
