PostGIS Connection Plugin
Defining PostGIS as a Data Source

The PostGIS data source connection is defined by a specific set of parameter.  It is accessed through the Data Source Selector.
To define PostGIS as a data source,  fill the Title and the minimal  Connection Information needed by PostGRES and PostGIS.
Open button will make available a set of datasets from the database specified in Database Name.



It is possible to Test the connection before open it. Also there is Advanced Connections Options to change same default parameters.
