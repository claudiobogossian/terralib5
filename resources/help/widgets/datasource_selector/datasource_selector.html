Data Source Selector

The Data Source Selector is used to Select an existing data source and make available its content as datasets, but  firstly it is used to define a way to connect with different data sources from where to search for data.

It is possible to add (+) a new connection, remove  (-) or edit (icon)  an existing connection. Also it is possible to create a  new (icon) empty Data Source establishing its name and connection parameters, if the data source offer creation as a capability.
  1. Microsoft Access 
  2. PostGIS
  3. Raster files (GDAL)
  4. Vector files (OGR)

Each type of Data Source has a set of parameters defined in an appropriate interface. If at least one data source is defined (PostGIS at this example), click on  Select to see all the datasets available. The dataset names will be showed and one or more datasets can be selected to be manipulated by the main window. 

It is important to note that the datasets still at the original data source. Also you can combine datasets from different sources, for example you can select  a shape file from a directory and a spatial table from PostGIS and draw them without any import/export action over the original data.

Click on Select  and a new interface will be provide with all datasets available. 

Defining new Connections to the chosen Data Source type

  1. Microsoft Access  and (+)
  2. PostGIS and (+)
  3. Raster files (GDAL) and (+)
  4. Vector files (OGR) and (+)

Click on  (icon-folder) to create a empty data source of the chosen type. For example create a new PostGIS data source called "my_database" and save everything it is created in your application there. For example, the result of an aggregation operation can be saved in "my_database", instead of save it in he original data source.

Click on (-) to  delete a previous defined connection to a data source. All datasets of the data source deleted will not be accessible by the Layer Explorer.

Click on (edit) to change  the parameters of  a  existent data source connection.

Creating a new Data Source

Click on (folder-icon) to create a new empty PostGIS data source and register it to the system. A new connection to the new data source must be defined in order to be able to save data on it.
