TerraView 5.0.0

TerraView is a Geographic Information System (GIS) that provides a friendly interface allowing you to view and manipulate both vector and raster data. This application was developed on top of TerraLib 5.0.0 geographic library which have been completely reengineered.

In this new version of TerraView some concepts previously used have changed and some new concepts were adopted. 

A short explanation of these concepts is presented here. To know about the GUI TerraView visit this link.

It also presents new (or enhanced versions of)  functions for data conversion and data exchanging; access to different data sources; map display; explanatory spatial data analysis; spatial and non-spatial queries; vector and raster processing and spatio-temporal analysis functionalities.
