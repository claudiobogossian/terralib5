Abrindo um Banco de Dados

 

Para abrir (ou conectar) um banco de dados, você deverá solicitar a opção banco de dados a partir do atalho , ou pelo menu Arquivo > Banco de Dados. Ao efetuar esta ação, a janela Banco de Dados será aberta e você terá a opção para criar um novo banco de dados ou conectar-se a um banco de dados já existente. No exemplo abaixo, você poderá verificar uma série de ações necessárias para abrir (conectar) um banco de dados existente.

1. Escolha a opção Conectar.

2. Defina o tipo de banco de dados a ser conectado. O TerraView pode utilizar vários SGBDs, como Access, Oracle, MySQL, etc... Selecione a opção Access.

3. Indique o diretório Bancos onde este banco de dados foi salvo.

4. Escolha o banco de dados a ser conectado.

5. Selecione Executar.

O Banco de Dados BH.mbd do seu diretório Tutorial/Bancos será aberto, como você pode visualizar na Figura abaixo.
