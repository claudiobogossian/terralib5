Fields Selection Dialog

This interface is used to configure which properties of a dataset will be used when linking two datasets and the join type. By default, all fields will be used and the the join type will be a left join,

The dialog contains two lists:
  •  The list of non-selected fields, which displays the fields that are not going to be included in the new layer;
  • The list of selected fields, which displays the fields that are going to be included in the new layer.

The dialog also contains a set of radio buttons used to select the join type to be used, the possible types are left join, join and right join.

How to use:
  1. Adjust the fields that are going to be used.
  2. Select the join type that is going to be used.
  3. Select on Properties the desired attributes and choose its type if the default is not adequate.
  4. Ok  - Closes the dialog and a adjust the fields and join type..
  5. Cancel  - It closes the dialog without applying any o f the changes.
