Vector to Vector

The vector to vector tool enables calculate and assign values ​​to a layer ("To" layer), based on information from another layer ("From" layer). Layers of these being any type of representation (polygons, lines or points).
Operations are available according to the type of representation and data types of the attributes. This makes that each type of data, and representation, has its possible operations.

The operations take place between all values ​​from "From" layer that the geometry have a certain spatial relationship with the "To" layer. The operation Below is a list of possible operations:

Value

This operation gets the value of "From" layer. If there are more than one intersection, the minimum value operation is used.

Statistical operations (Minimum value, Maximum value, Mean, etc...)

This operation calculates some statistical operations.

Class with highest occurrence

This operation calculates the amount of times a class occurs in a relationship. Using the figure as example, in this case the class with occurence is de "Warehouse". Becouse there are four ouccurrence of "Warehouse" against two of "Soya" and only onde of "Corn".

Class with highest intersection area

This operation calculates which class is predominant in relation to area. It can be observed that the polygon "Corn" has the largest intersection with the farm, but the sum of the areas of "Soya" intersections is greater than the "Corn". This makes "Soya" is predominantly.

Percent per class

This operation calculates the percentage of the classes in relation. Using the example, because there are four classes, each class receive 0.25.

Minimum distance

This operation calculates the minimum distance of all geometries from "From" layer which do not intercept "To" layer geometries.

Presence

This operation check whether at least one "From" layer geometry intercepts a "To" layer geometry.

Weighted by area

This operation claculates the weighted average by area.

Weighted sum by area

This operation claculates the weighted average sum by area.

Percent of each class by area

This operation calculates the percentage of class area and the intersect area.

Percent of total area

This operation claculates the percentage of class area and "To" geometry total area.

  • Choose spatial relation available
  • Select the output repository
    • File ou Datasource
    • Output name
  • Select the desired operations
