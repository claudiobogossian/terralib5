Creating a  new empty PostGIS  database as Data Source

To create a new PostGIS  database as data source it is necessary to inform  the Server information (Host, Port, User, Password) and also the Creation Information (New Database Name and the Template used (template_postgis_20).  Advanced  Options is used to change same default parameters, like Owner, Encoding, TablespaceLimit connections whose default values are Postgres, UTF8, pg_default and 4  respectively.
and

After that it is necessary to define a new PostGIS connector to  this  database (my_database_test)  in order to be able to save data on it. See how to create a PostGIS connector here.
