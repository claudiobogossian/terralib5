Microsoft Access (ADO) Connector 
Defining Microsoft Access (ADO) as a Data Source

The  Microsoft Access (ADO) Connector is used to define a connection with a specific  database. It is accessed through the Data Source Selector.

To define Microsoft Access (*.mdb) database as a data source in your application,  fill the Title, the Connection Information Provider (is automatically filled) and File/URL with the path (clicking on ...) to the data source (*.mdb ), and Password  when applicable.




Open will make this connection available at Data Source Selector with the same title to be selected.
