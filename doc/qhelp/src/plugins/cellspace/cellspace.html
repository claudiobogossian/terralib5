Cellular Spaces

The Cellular Spaces is a tool that allows the creation of a cellular dataset that can be used in various ways.

At first the goal is to homogenize information from different sources, in different formats (vector data, raster and also other cell spaces), adding them in the same base.

To create a cellular space use the option:

Processing > Cellular Spaces > Create Cellular Space...

If there is layer with SRID in the tree, the cellular space can be created having a layer reference.

Simply fill in the fields:


  • Referênce Layer: if there is, a reference layer may be chosen from the list.

  • Use Mask or not: using the layer as a mask, will only be created cells that intersect any geometry layer.

  • Bounding Box: these fields are initialized with the reference layer information. If necessary, this information can be changed.

  • Cellular Space Type: there are two options for creating the layer of cells;

    • Polygons: will be created geometries of polygon type to represent the cells;

    • Points: will be created geometries of point type to represent the cells;

  • Resolution or number of columns and rows: can opt to choose the resolution of the cell or the number of columns and rows of cellsthat will be created.

  • Output: select the location where the dataset is generated;

    • Repository: file or DataSource;

    • DataSet name.

After the creation, the option to create the layer in the tree is displayed.
