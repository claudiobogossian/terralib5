How to manipulate data?

Operations for inclusion of data in the application:

  • Data Source Selector
  • Dataset Selector

Operations to exchange data between data sources:

  • Data Source Exchanger
  • Layer Exchanger

Operations on the data in the application:

  • Classification
  • Charts
  • Pie Bar

Operations on the data table:

  • Data Table

Operation to set / change the spatial reference system  of a dataset:

  • Spatial Reference System

Types of layers: 

  • Dataset Layer: Uses a Data Source Selector and Dataset Selector to create a Dataset Layer.
  • Query Layer
  • Tabular File
