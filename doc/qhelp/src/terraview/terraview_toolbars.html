Tool Bars

The default tool bars from TerraView are:

  • Project
    • New
    • Open
    • Save
  • View
    • Layer Explorer
    • Map Display
    • Data Table
    • Style Explorer
  • Map Tools
    • Draw
    • Undo
    • Redo
    • Zoom Extent
    • Zoom In
    • Zoom Out
    • Pan
    • Information
    • Selection
Through the Tool Bar Configuration is possible to change the toolbars.
