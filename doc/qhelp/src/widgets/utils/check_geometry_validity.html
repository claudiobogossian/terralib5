Check Geometry Validity

It is accessed through:

Tools > Check Geometry Validity...

The Check Geometry Validity is used to verify all geometries of a layer and check if the geometry is valid.
  1. Select the layer that will be analyzed (Only vector layers).
  2. Click on Verify button to begin checking. 
  3. With the layer drawn, use double-click at the invalid geometries items to see the position on Map.
