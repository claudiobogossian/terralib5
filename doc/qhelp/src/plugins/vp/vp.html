Vector Processing Plugin

The Vector Processing plugin contains several processing algorithms applied to vector data. Such as Aggregation, Arithmetic Operations, Buffer and Intersection.

  •  Aggregation - The Aggregation operation is applied to a single layer and a chosen group of attributes that can be summarized. The idea is to generate a new layer with summarized information. The geometries of each group are aggregated through the UNION spatial operation and the attributes can be summarized using of the aggregation operation: sum, count, maximum, minimum, average and so on. 
  • Buffer - The Buffer is a spatial operation that can be applied to a single layer. The buffer zone may be defined as an area generated around an object with a predetermined distance.
  • Intersection - The Intersection is spatial operation and can be applied for two layers. One containing polygons that form a clipping mask, and another (with any kind of representation ex. polygons, lines, points) will be clipped. The result is a new plan formed by the objects of the second theme in the intersection with the clipping mask.
