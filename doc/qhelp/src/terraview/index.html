TerraView 5.0.0

TerraView is a Geographic Information System (GIS) that provides a friendly interface allowing you to view and manipulate both vector and raster data. This application was developed using TerraLib 5.x geographic library.

In this new version of TerraView some concepts previously used have changed and some new concepts were adopted. A short explanation of these concepts is presented here. To know about the GUI TerraView visit this link.
